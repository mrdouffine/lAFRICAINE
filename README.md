# LAFRICAINE - Menu Sename

Site web minimaliste reproduit en Next.js.

## Caractéristiques

- Design minimaliste en noir et blanc
- Menu avec séparateurs en points
- Typographie Helvetica Neue
- Responsive design
- Liens sociaux (LinkedIn, Twitter)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur [http://localhost:3000](http://localhost:3000)

## Build Production

```bash
npm run build
npm start
```

## Structure

- `app/page.js` - Page principale avec le menu
- `app/layout.js` - Layout racine
- `app/globals.css` - Styles globaux
